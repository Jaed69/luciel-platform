**GET /dashboard/saldos** query:
```
SELECT cuentas.codigo, cuentas.nombre, cuentas.moneda,
       SUM(asiento_lineas.debe) AS total_debe,
       SUM(asiento_lineas.haber) AS total_haber,
       SUM(asiento_lineas.debe) - SUM(asiento_lineas.haber) AS saldo
FROM cuentas
JOIN asiento_lineas ON asiento_lineas.cuenta_id = cuentas.id
JOIN asientos ON asientos.id = asiento_lineas.asiento_id
LEFT JOIN tours_servicios ON tours_servicios.asiento_id = asientos.id
WHERE asientos.fecha BETWEEN :fecha_desde AND :fecha_hasta
  AND (:agencia_id IS NULL OR tours_servicios.agencia_id = :agencia_id)
  AND (:vendedor_id IS NULL OR tours_servicios.vendedor_id = :vendedor_id)
  AND (:moneda IS NULL OR cuentas.moneda = :moneda)
GROUP BY cuentas.id
```
Cuando moneda filter es 'PEN' o 'USD' aplica; cuando 'Ambas' / NULL → no filtra. Maneja tours_servicios NULL (asientos TC interno no tienen tour_servicio asociado, pero suman al saldo de caja).
