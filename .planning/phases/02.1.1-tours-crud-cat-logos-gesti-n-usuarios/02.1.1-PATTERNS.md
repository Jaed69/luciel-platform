### 1. `PUT /catalogos/{entidad}/{id}` (catalog edit)

**Analog:** `PUT /cuentas/{cuenta_id}` — `apps/tours/api/app/routers/core.py` lines 64-80.

**Why exact:** Both are generic-PUT-by-id with `select` → mutate → `commit` → `refresh` → return. D-02 says "sigue el mismo patrón que `PUT /cuentas/{cuenta_id}` (line 64)". Only difference: PUT catalogos uses `CatalogoIn` (codigo + nombre, no tipo/moneda) and the RBAC dependency switches to `require_role("admin", "contabilidad")` (D-13).

**Mold** (`core.py` lines 64-80):
```python
@router.put("/cuentas/{cuenta_id}", response_model=CuentaOut)
async def update_cuenta(
    cuenta_id: int,
    body: CuentaIn,
    session: AsyncSession = Depends(get_session),
    _user: dict = Depends(require_role("admin")),
) -> Cuentas:
    cuenta = (await session.execute(select(Cuentas).where(Cuentas.id == cuenta_id))).scalar_one_or_none()
    if cuenta is None:
        raise HTTPException(status_code=404, detail="Cuenta no encontrada")
    cuenta.codigo = body.codigo
    cuenta.nombre = body.nombre
    cuenta.tipo = body.tipo
    cuenta.moneda = body.moneda
    await session.commit()
    await session.refresh(cuenta)
    return cuenta
```

**Apply (D-02 + D-03):**
```python
# D-13: admin OR contabilidad (D-24 was admin-only — D-24 modified by 02.1.1)
@router.put("/catalogos/{entidad}/{row_id}", response_model=CatalogoOut)
```

**Reference count map** (D-19, planner refines the columnas — principle: "si hay historia contable, no se borra"):
```python
# from models/tours.py lines 111-125 — ToursServicios has FK columns to every catalog
# D-19 default rule: tours_servicios WHERE activo=1 counts as a reference
# Plus: monedas → cuentas (101-CAJA-PEN/USD cannot be deactivated if cuentas exist)
# (search for "tours_servicios.tour_id", "tours_servicios.agencia_id" etc.)

_REFERENCED_BY: dict[str, list[tuple[str, str]]] = {
    "agencias":     [("tours_servicios", "agencia_id")],
    "vendedores":   [("tours_servicios", "vendedor_id")],
    "tours":        [("tours_servicios", "tour_id")],
    "formas-pago":  [("tours_servicios", "forma_pago_id")],
    "monedas":      [("tours_servicios", "moneda_id"), ("cuentas", "moneda")],  # enum column, special
}
```

**Apply (D-18 — block with 409 + referencias list):**
```python
from sqlalchemy import func, text

@router.delete("/catalogos/{entidad}/{row_id}")
async def delete_catalog(entidad, row_id, session, _user=Depends(require_role("admin", "contabilidad"))):
    model = _CATALOG_MODELS.get(entidad)
    if model is None:
        raise HTTPException(404, f"Catálogo '{entidad}' no existe")
    row = (await session.execute(select(model).where(model.id == row_id))).scalar_one_or_none()
    if row is None:
        raise HTTPException(404, "Registro no encontrado")

    # D-18 referential check
    refs: list[dict] = []
    for tabla, columna in _REFERENCED_BY.get(entidad, []):
        if columna == "moneda" and entidad == "monedas":  # enum on cuentas
            count = (await session.execute(
                select(func.count()).select_from(text("cuentas")).where(
                    text("cuentas.moneda = :val AND cuentas.activo = 1").bindparams(val=row.codigo)
                )
            )).scalar_one()
        else:  # FK on tours_servicios
            count = (await session.execute(
                select(func.count()).select_from(text(tabla)).where(
                    text(f"{tabla}.{columna} = :id AND {tabla}.activo = 1").bindparams(id=row_id)
                )
            )).scalar_one()
        if count > 0:
            refs.append({"tabla": tabla, "count": int(count)})

    if refs:
        total = sum(r["count"] for r in refs)
        raise HTTPException(
            status_code=409,
            detail={"mensaje": f"No se puede desactivar — {total} registros lo referencian", "referencias": refs},
        )

    row.activo = False
    await session.commit()
    return {"ok": True}
```

**Why this shape:** SQLAlchemy ORM `select(Model).where(...)` doesn't naturally express "count on raw table by FK" without joining the FK table class. The 5 catalog tables have FK only in `tours_servicios`; counting active references is one `COUNT(*)` per `(tabla, columna)` pair. The `text()` form keeps the check explicit and matches D-19's "tabla/columna" map. (Avoid introducing a `referencias.py` service for one DELETE handler — `ponytail: inline check, extract if a second domain needs the same logic`.)

```python
def require_role(*roles: str):
    """Dependency factory — 403 if the authenticated user's role is not in `roles`."""
    async def _checker(user: dict = Depends(get_current_user)) -> dict:
        if user["role"] not in roles:
```

```ts
const API_URL = process.env.TOURS_API_URL || "http://tours-api:8000";

export async function POST(req: Request) {
```

```ts
const API_URL = process.env.TOURS_API_URL || "http://tours-api:8000";

export async function proxyJson(path: string, method: string, body?: string): Promise<Response> {

  });

}
```
