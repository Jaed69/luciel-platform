<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Internet → Traefik | All external traffic enters here. TLS termination, HTTP-01 challenges, routing. |
| Traefik → Docker socket | Traefik reads Docker API for label discovery. Must be read-only. |
| Traefik → acme.json | Certificate storage. Contains private keys in base64. |
| Traefik → landing container | Reverse proxy to nginx:8080. Internal Docker network only. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-01-01 | Tampering | Docker socket mount | critical | mitigate | Mount /var/run/docker.sock:ro — Traefik only needs read access for label discovery. Write access would allow container escape. |
| T-01-02 | Elevation | Traefik container | high | mitigate | Run as non-root user 65532:65532 + security_opt no-new-privileges:true. Prevents privilege escalation even if Traefik is compromised. |
| T-01-03 | Information Disclosure | acme.json | high | mitigate | chmod 600 + chown 65532:65532. File contains private key material. Gitignored to prevent accidental commit. |
| T-01-04 | Information Disclosure | Traefik dashboard | medium | mitigate | basicAuth middleware with bcrypt-hashed credentials from .env. HTTPS-only (websecure entrypoint). Dedicated subdomain traefik.luciel.dev. |
| T-01-05 | Denial of Service | LE rate limits | medium | mitigate | Start with staging CA (D-12). Verify routing works before flipping to production. Staging has 50 certs/week vs production 50 certs/week per registered domain. |
| T-01-06 | Spoofing | HTTP traffic | medium | mitigate | Traefik web entrypoint forces permanent HTTP→HTTPS redirect. No content served over plain HTTP. |
| T-01-07 | Information Disclosure | .env file on VPS | medium | mitigate | .env gitignored. File permissions 600. VPS access restricted by ufw (22/80/443 only) + fail2ban for SSH brute-force protection. |
| T-01-SC | Tampering | npm/Docker image installs | high | mitigate | All packages audited in RESEARCH.md Package Legitimacy table (all [OK]/Approved). Docker images are official (traefik, node, nginx) with verified arm64 manifests. |
</threat_model>
